# LibAA
LibAA is a audio library.

## Audio Effect
+ Biquad Filter
+ Chorus
+ Compressor
+ Delay
+ Distortion
+ Flanger
+ Tremolo

## DSP
+ DTW
+ MFCC
+ YIN
+ FFT
+ STFT

## Audio Codec
+ Decoder
  + Wav
  
# How to build libaa
```bash
cmake -S . -B build
cmake --build build
```
